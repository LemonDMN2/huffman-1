# Huffman Coding Algorithm

    $ ./huffman

    Huffman coding algorithm
    by Sergey Tikhonov (st@haqu.net)

    Usage: huffman [OPTIONS] input [output]
      The default action is to encode input file.
      -d	Decode file.

    Examples:
      huffman input.txt
      huffman input.txt encoded.txt
      huffman -d encoded.txt
